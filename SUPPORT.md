# Support

If you need help with the SpeakEase project, you can reach out through the following channels:

## Opening an Issue
If you encounter a bug, have a feature request, or need help setting up the project locally, please [open an issue](../../issues) on our GitHub repository. 

When creating an issue, please try to include as much detail as possible to help us resolve it quickly:
- Steps to reproduce the bug
- Expected behavior vs. actual behavior
- Your environment details (OS, Node version, Browser, GPU)
- Screenshots or console logs if applicable

## Contact the Maintainer
