# Security Policy

## Supported Versions

Only the latest version of SpeakEase is supported for security updates.

## Reporting a Vulnerability

Security is a top priority for this project. If you discover a security vulnerability, please do not disclose it publicly.

Instead, please report vulnerabilities by opening a private security advisory on GitHub or by contacting the maintainer directly if contact details are provided in the repository.

### What to Include

1. A detailed description of the vulnerability.
2. The components affected.
3. Steps to reproduce or a proof of concept.
4. Potential impact.

### Our Response

The maintainer will acknowledge receipt of the report within 48 hours and provide a timeline for investigation and resolution.
